--An HTML editing tool: jsfiddle.net/WW3bh
--World Wide Web consists of a number of HTML documents connectgeed by HyperLinks.
--HTML markup is made of "tags" of the form: <name>
each of which forms an "element" of the form: <name>contents </name>
Some tags:
"inline" tag: renedered inline with preceding and following text
<b>
<em> (for italics rendering)
following 2 tags are also "void" tags, i.e., do not need closing tags
<a> ("anchor" used for making links)
<img>
<br>
"block" element: rendered as a box of a given width and height
<p>
"container" element:
<span> (it is an "inline" tag)
<div> (it is a "block" tag)

Attributes are properties of tags:
<img href="mypic.gif" alt="mypic"> (href, alt are attributes)
<div class="myclass" > (class is an attribute may be used for CSS)

--multiple spaces, tabs, newline are rendered to a single space

Structure of HTML document:
<!DOCTYPE html>
<html>
<head>
...has metadata about the document...
<title>...</title>
...
</head>
<body>
..."contents" of the "document" with tags; subject of this course mostly this
</body>
<html>
